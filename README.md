# 🤔 Secret Word

## 🌐 Projeto Web

Site desenvolvido com React JS, onde o mesmo se trata de um jogo semelhante ao Roda a Roda do SBT, onde o usuário receberá uma dica da palavra e tentará adivinhar a palavra correta.

### 🛠️Construído com

* React JS;
* HTML;
* CSS.
